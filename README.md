# Ragnarok Boss Respawn Reminder Bot

Телеграм-бот для отслеживания времени убийства боссов и напоминания об окне респавна.
Многопользовательский: каждый, кто пишет боту, ведёт свои личные таймеры — они не пересекаются с другими игроками.

## Локальный запуск

1. Создай бота через `@BotFather` в Telegram, получи токен.
2. Скопируй `.env.example` в `.env` и впиши туда токен:
   ```
   TELEGRAM_BOT_TOKEN=твой_токен
   ```
3. Установи зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Запусти бота:
   ```
   python bot.py
   ```
5. Напиши боту `/start` в Telegram — он запомнит чат для рассылки напоминаний.

## Команды

- `/list` — список боссов и их коды
- `/kill <код> [ЧЧ:ММ]` — отметить убийство босса (без времени — берётся текущий момент)
- `/status` — состояние всех боссов
- `/cancel <код>` — отменить отметку и запланированные напоминания

## Боссы

Список и респавны заданы в `bosses.json` — можно редактировать без изменения кода.

## Деплой на Railway (работает 24/7, без твоего компа)

1. Зайди на [railway.app](https://railway.app), залогинься через GitHub.
2. New Project → Deploy from GitHub repo → выбери `Ajwata/ragnarok`.
3. В настройках сервиса (Variables) добавь переменную `TELEGRAM_BOT_TOKEN` со значением токена от `@BotFather`.
4. Railway сам определит `Procfile` и запустит `worker: python bot.py`.
5. Чтобы данные (`data.json` — отметки всех пользователей) не терялись при редеплое, добавь Volume (Settings → Volumes), примонтируй его, например, на `/data`, и добавь переменную `DATA_DIR=/data`.
6. После деплоя открой логи сервиса — там должна появиться строка `Bot starting...`. Дальше бот доступен всем в Telegram по его @username, независимо от того, включён ли твой компьютер.

## Заметки

- `data.json` создаётся автоматически и не должен попадать в git (уже в `.gitignore`).
- Данные хранятся по `chat_id` — у каждого пользователя/чата свои отдельные отметки и напоминания.
- Бот держит соединение открытым (`run_polling`), пока запущен процесс.
- При перезапуске бот подхватывает уже отмеченных боссов у всех пользователей из `data.json` и пересоздаёт напоминания, если окно ещё не прошло.
